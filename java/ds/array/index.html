# Array란?

# 초기값

자바에서는 할당만 하고 초기화하지 않았을때는 type에 따라 초기값이 설정된다.

  • int[] 초기값: 0
  • double[] 초기값: 0.0
  • String[] 초기값: null
  • boolean[] 초기값: false

# 배열의 한계점 / 장점

  • 할당을 하고난후에는 크기가 고정적이다.
  • 추가,삭제의 비용이 크다.
  • 기능이 없음
  • (장점)인덱스가 있다. 빠른 조회 가능.
  • (장점)데이터를 순차적으로 나열할수있다. 성능 향상에 도움이됨.

배열의 장점이 단점을 커버할수있는 경우가 아니라면 List를 쓰는 것이 좋아보인다. 자바 util 패키지에서 ArrayList와 LinkedList라는 List 를 지원한다.

(* 엄밀히 말하면 ArrayList와 LinkedList는 트리로 구현이 되어있다.)
