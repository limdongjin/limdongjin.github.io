# vue-progressive-image + spinner

이미지가 렌더링 되기전에 spinner를 돌릴수있다.

<progressive-img:src="post.url">
    <div slot-scope="{ visible }" v-if="visible">
        <atom-spinner
            :animation-duration="1000"
            :size="60"
            :color="'#ff1d5e'"/>
    </div>
</progressive-img>
