# 코드조각. html형식의 string 변수를 View에 그려주기

@html_data = "<h1> Hi. <p> hello</p> </h1>"
1

@html_data 와 같은 루비 변수가 있을때 이를 html.erb 에서 단순히 <%= @html_data %>로 쓰게되면 html태그가 그대로 출력되게된다. 이를 해결하기위해서는

<%= @html_data.html_safe %>
1

.html_safe 를 써주면 해결할수있다!!
